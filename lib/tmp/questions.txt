1,O que é um algoritmo?,Um algoritmo é uma sequência finita de passos bem definidos para resolver um problema.

3,O que caracteriza a programação orientada a objetos?,A programação orientada a objetos é baseada em classes e objetos, utilizando conceitos como herança, encapsulamento e polimorfismo.
3,Qual é a diferença entre programação imperativa e declarativa?,A programação imperativa foca em como resolver o problema, enquanto a declarativa foca no que deve ser resolvido.
3,O que é programação funcional?,A programação funcional é baseada em funções matemáticas e evita mudança de estado.
4,O que é a arquitetura von Neumann?,É um modelo de arquitetura onde dados e instruções são armazenados na mesma memória
4,Qual a função do barramento do processador?,O barramento conecta diferentes componentes do computador, permitindo a troca de dados.

5,O que são registradores?,Registradores são pequenas memórias de alta velocidade dentro do processador usadas para armazenar dados temporários.

5,O que é a unidade de controle de um processador?,A unidade de controle gerencia a execução das instruções do processador.
6,O que é um sistema operacional?,Um sistema operacional é um software que gerencia o hardware e fornece serviços para programas de aplicação.
6,O que é um processo?,Um processo é uma instância de um programa em execução.

7,O que é um endereço IP?,Um endereço IP é um identificador numérico único atribuído a um dispositivo em uma rede.
